从0开始 用企业微信做一个属于你的定时天气推送

  • 2021-05-30

成品如图

它也可以是这样的

两种只是代码有略微的区别而已，功能都是一样的。

只需要五个工具：

1.企业微信账号 https://work.weixin.qq.com/wework_admin/register_wx?from=myhome
2.腾讯云函数（可以白嫖每个月免费额度很多） https://cloud.tencent.com/product/scf
3.和风天气 https://id.qweather.com/#/register?redirect=https%3A%2F%2Fconsole.qweather.com
4.天行数据(图文界面并没有添加此功能，所以可以忽略) https://www.tianapi.com/
5.一双灵活的双手以及脑袋

教程开始：

1.首先打开企业微信官网注册一个账号：
随意填写，管理员微信用自己的。

在图中的位置记住自己的企业id
再点击上面的应用管理，找到自建->创建应用，名称logo随便填写，下面的可见范围选择自己的部门或者成员，应用创建好后点进去最上面可以看见应用的Agentid，以及secret，secret貌似需要下载企业微信查看。（应该还有一步添加企业成员，自己添加一下就可以了，这里就不放教程了）。
有了企业id，应用Agentid，secret开始我们的下一步。
2.登录腾讯云函数
点击左侧函数服务

自定义创建，名称地域随意，环境选择python3.6。
拉到最下面触发器配置，
触发方式定时触发，定时触发实现在固定的时间执行我们的代码，触发周期可以用默认的，也可以自定义，自定义表达式可以点击下面的cron相关文档查看，例如在每天的上午8点和下午两点执行代码可以设置为0 0 8,14 * * * *
点击完成
3.下一步获取和风天气key
注册登录好之后点击进入 和风天气开发者控制台
应用管理->右上角创建应用->免费个人->名称随便填写->webapi->填写key名称（此处只是名称不是key）->完成后复制key
4.从天行数据获取文案(图文界面没有此功能)
天行数据注册登录后点击三个链接申请下接口：
https://www.tianapi.com/apiview/198
https://www.tianapi.com/apiview/194
https://www.tianapi.com/apiview/143
来到控制页 https://www.tianapi.com/console/ 点击数据管理，生成APIKEY拿到key

5.最后修改我们的代码

文案接口的代码，下面是没有文案接口的图文代码

修改位置信息在value1 = { # 位置 location处修改即可，注意此处还需要用到和风 >天气key 将其填写到4个key中
也可以添加其他天气信息，需要用到和风天气api
最后在main函数处将corpid = “更改为你的企业id”、secret = “更改为你的应用>secret”、agentid = “你的应用Agentid”。
如果需要文案接口在key处填写即可。

6.将我们修改好的代码放到腾讯云函数
进入腾讯云函数，函数管理，函数代码将我们的代码全选复制进去点击左下角的部署并测试查看执行摘要，测试成功，即可在企业微信收到推送消息了。
